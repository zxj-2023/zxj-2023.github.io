git连接远程仓库

初始化仓库：git init

目标 命令
查看远程仓库 git remote -v
修改远程仓库地址 git remote set-url origin <新地址>
添加新远程仓库（不同名） git remote add upstream <新地址>
删除远程仓库 git remote remove origin
添加第一个远程仓库 git remote add origin
