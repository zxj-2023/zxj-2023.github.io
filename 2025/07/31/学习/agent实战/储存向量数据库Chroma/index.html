Chroma是什么

Chroma（通常指 ChromaDB）是一款 开源、AI 原生的向量数据库，专为存储和检索 高维嵌入向量 而设计，目标是让开发者 5 分钟内在本地跑起一个语义搜索或 RAG 系统。

极简安装：pip install chromadb

双运行模式：

  • 内存模式：调试/原型随意重启；
  • 持久化模式：指定 persist_directory 即可落盘，生产也不怕丢数据。

HNSW 索引：百万级向量也能 毫秒级 响应。
